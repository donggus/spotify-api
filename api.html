/songs

Collection of available songs

/songs post

post

Get a list of songs based on the artist name.

post /songs

Get a list of songs based on the artist name.

  • Request
  • Response

Query Parameters

  • artist: required(string - minLength: 3)

    The name of the artist (case insensitive and doesn't need to match the whole name of the artist)

    Example:

    Red Hot Chili

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

{
  "artist": "RED HOT CHILI PEPPERS",
  "songs": [
      "Otherside",
      "Scar Tissue"
  ],
  "next": {
      "page": 4,
      "last_page": 5
  },
  "previous": {
      "page": 2
  }
}
